JS文件格式统一为UNIX格式(\n)；

统一使用TAB缩进，TAB除了用于缩进之外不允许用于其他用途（这意味着每一行第一个非TAB字符之后不再出现TAB）；

每条语句后面的单行注释前面保留两个空格；

HTML中不出现一行JS代码，只允许出现在script标签内或者通过script标签引用外部JS文件；

除个别变量名($,$E等)，禁止使用$符号作为变量名的组成字符（包括$开头的变量名）；

变量的命名宏观上遵循类Java的命名规则：驼峰命名法（Camel-Case）；

局部变量不使用下划线开头，类的私有成员变量，全部以下划线开头；

包的组织结构和java保持一致；

永远不显示的使用undefined，因为这些地方都可以使用null取代之；

所有用new Object(),new Array()的地方使用{},[]替代；

&&,||,&,|等表达式不作为单独的语句出现，=,+=等赋值语句不出现在条件表达式中；

for循环中任何重复性的操作符运算都应该使用一个局部变量替代；

使用数组保存顺序，使用对象进行哈希映射（快速查找）；

使用for-i遍历数组，使用for-in遍历对象，不要颠倒使用；

不要把数组当成对象使用哈希映射功能；

for-i循环变量只使用i,j,k三个，for-in推荐循环变量k,key；

函数内循环不超过三层，如果达到三层，请考虑将部分功能拆分到另一函数中；

“{”（左大括号）应在复合语句起始行的结尾处；

“}”（右大括号）应与“{”（左大括号）的那一行的开头对齐；

参考python语法，加强通过缩进识别语句块的习惯；
